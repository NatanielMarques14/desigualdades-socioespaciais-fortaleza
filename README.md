# Análise das Desigualdades Socioespaciais em Fortaleza

Projeto desenvolvido para a disciplina CK0441 – Laboratório de Ciência de Dados.

## Objetivo

Investigar a distribuição espacial de indicadores demográficos, educacionais e habitacionais nos setores censitários de Fortaleza, utilizando dados públicos do Censo Demográfico 2022 e técnicas de Ciência de Dados e Geoprocessamento.

## Tecnologias

- Python
- Pandas
- GeoPandas
- Matplotlib
- Seaborn

## Fonte dos Dados

- Censo Demográfico 2022 (IBGE)
- Malha de Setores Censitários do IBGE
